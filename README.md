### Install Packages

    install.packages(c("tidyverse", "tidytext", "janeaustenr", "wordcloud2"))

### Process

1.  Get data

2.  Wrangle Data

3.  Tokenize corpus

4.  Generate word clouds

5.  Join with sentiment dictionaries

6.  Visualize sentiment
